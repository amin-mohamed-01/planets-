Executive Summary

This AI system provides accurate vegetation mapping and early detection of crop stress and irrigation deficiency. By combining deep learning with remote sensing spectral indices, Desert Agriculture Sentinel delivers high-precision vegetation masks, health classification, and time-based environmental change analysis.

The tool supports farmers, environmental organizations, and government agencies in improving resource allocation, reducing water loss, and increasing agricultural productivity in desert regions.

Crop and Water Health Monitoring in Desert Agriculture using ML Models & Satellite Images

Strategic Impact

  • Early detection of agricultural stress and irrigation shortages
  • Improved water management in desert farms
  • Monitoring crop health for government agricultural programs
  • Supporting "Sustainable Agriculture 2030" initiatives
  • Helping farmers reduce losses and optimize irrigation schedules
  • Providing geospatial insights for large-scale farming corporations

Key Technologies

  • Deep Learning segmentation using U-Net (ResNet34 encoder)
  • Classical vegetation indices: NDVI, GNDVI, NDWI, SAVI, MSI
  • Multi-stage stress detection logic
  • Gaussian filtering for noise reduction
  • Temporal change detection (before/after images)
  • Full agricultural health reporting

Key Achievements

Major Accomplishments
  • Implemented a fully functional vegetation detection system using AI + classical indices
  • Integrated U-Net semantic segmentation for crop boundaries and canopy detection
  • Computed advanced vegetation indices (NDVI, GNDVI, NDWI, SAVI, MSI)
  • Built automatic stress/water-deficit detection using NDWI + NDVI thresholds
  • Designed a Streamlit dashboard for real-time visualization
  • Added change detection between before/after images
  • Generated downloadable agricultural health reports (CSV)
  • Supported GeoTIFF satellite imagery through Rasterio
Major Challenges Overcome
  • Handling multispectral bands and normalizing Sentinel/Landsat imagery
  • Ensuring U-Net inference runs smoothly on CPU without GPU
  • Balancing NDVI/NDWI thresholds across different crop types
  • Distinguishing stressed crops from naturally low-vegetation desert backgrounds
  • Managing large satellite images within memory limits
Extra Attributes Added by Engineer
  • Automatic fallback to classical indices if U-Net fails
  • Multi-index stress detection framework
  • RGB-only NIR estimation for non-multispectral images
  • Streamlit visualization with color-coded masks
  • Temporal change detection engine
  • Unified CSV report generator
  • Multi-language support (English/Arabic)
  • Mobile-responsive interface design

Dataset Information

Data Sources

Technical Specifications

  • Size: 10 MB to 300 MB per scene depending on resolution
  • Format: GeoTIFF (preferred), TIFF, PNG, JPG
  • Compatibility: Fully supports Rasterio for TIFF/GeoTIFF and PIL/OpenCV for RGB images
