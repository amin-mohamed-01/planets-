Project Overview

Project Description

This project provides an advanced artificial intelligence system for detecting and tracking sand dune encroachment and desertification using satellite imagery such as Sentinel-2, Landsat, and drone images.

The application combines:

  • Deep Learning semantic segmentation (U-Net with ResNet34 encoder)
  • Classical spectral sand indices (SI, BI, NDSI, RI)
  • Gaussian filtering & threshold-based sand detection
  • Temporal change detection between "Before/After" satellite images
  • Automated report generation

The solution was deployed using Streamlit as an interactive monitoring tool.

Executive Summary & Achievements

Key Achievements

  • Developed a full functional AI-powered sand detection system
  • Integrated U-Net deep learning segmentation with real satellite imagery
  • Added fallback classical analysis when PyTorch is unavailable
  • Designed a user-friendly Streamlit interface
  • Implemented before/after temporal analysis to measure dune movement
  • Generated automated CSV reports with risk assessment
  • Supported high-resolution GeoTIFF (Rasterio) satellite datasets

Major Challenges

  • Processing large GeoTIFF satellite images efficiently
  • Ensuring the deep learning U-Net model runs on CPUs without GPU
  • Balancing sand-detection thresholds for different environments
  • Handling multi-spectral bands from Sentinel-2 and Landsat
  • Reducing false positives in bright non-sand regions (e.g., urban, salt flats)

Strategic Impact & Business Model

Strategic Impact

  • Provides government agencies with early warnings of desertification
  • Enables environmental decision makers to track desert expansion
  • Supports agricultural planning and land protection policies
  • Helps national sustainability programs (e.g., Egypt 2030)
  • Can be scaled to monitor coastal erosion or vegetation loss

Business Model Transformation

This system can be commercialized as:

  1. 1 Environmental Monitoring Service - Providing monthly reports for desertification risk per region
  2. 2 SaaS Dashboard for Government / NGOs - Online portal for satellite-based monitoring
  3. 3 Integration with Smart Agriculture - Protecting farms from dune encroachment
  4. 4 Remote Sensing Consultancy Tool - Selling insights to companies (oil, mining, infrastructure)

Technical Details

Dataset Information

  • Dataset: Sentinel-2 (10 m RGB), Landsat-8 (30 m RGB), UAV drone images from open datasets
  • Size: Average 10–250 MB per scene (tiff)
  • Format: TIFF, GeoTIFF, JPEG, PNG
  • Compatibility: Fully compatible with Rasterio, OpenCV, PIL, and Streamlit
